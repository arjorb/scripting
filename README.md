# Shell Scripting Collection

This repository contains a set of simple shell scripts:

- [`api_call.sh`](api_call.sh): Script for making API calls.
- [`interactive_menu.sh`](interactive_menu.sh): Interactive menu example.
- [`password_generator.sh`](password_generator.sh): Generates random passwords.
- [`remote_check.sh`](remote_check.sh): Checks remote server status.
- [`status.sh`](status.sh): Displays system status information.

Each script is standalone and can be run directly in a Unix-like shell.
